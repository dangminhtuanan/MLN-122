Put background music files in this folder.

Required names:
- world-cup-football.mp3
- fishing-chill.mp3

Notes:
- Use music you own or have permission to use.
- The football game automatically uses `world-cup-football.mp3` as the World Cup style background music.
- The fishing game automatically uses `fishing-chill.mp3` as the relaxing chill background music.
- Browser autoplay is blocked, so players still need to press "Phát nhạc" once.
